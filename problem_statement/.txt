# Public Transport Delay Prediction Project

## Problem Statement

Public transport systems often experience delays due to several factors such as weather conditions, traffic congestion, public events, and peak travel hours. These delays affect passenger travel planning and reduce transport efficiency.

The goal of this project is to analyze these factors and build a machine learning model that predicts **how many minutes a transport trip will be delayed**.


Target variable:
delay_minutes 

---

## Objectives

* Clean messy real-world transport data.
* Analyze how weather, traffic, and events influence delays.
* Build regression models to predict delay time.
* Compare models and select the best performing model.
* Provide insights that can help improve transport scheduling.

---
